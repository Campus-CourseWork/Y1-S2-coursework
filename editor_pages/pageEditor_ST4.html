Editor Page - Stage 4

My Role and Contributions

I am Student 4 and responsible for creating the Profile page, the Sitemap page, and the Content ST4 page on Affordable Housing & Inclusion. I designed and coded these pages ensuring accessibility, responsiveness, and semantic markup.

Technical Description

The pages utilize HTML5 semantic elements, CSS Grid and Flexbox layouts, and follow a sustainable green theme consistent with the project's branding. I used ARIA roles and labels for improved accessibility and ensured responsive navigation with mobile menu toggles.

Accessibility and Compliance

All pages were tested for keyboard navigation and screen reader compatibility. Color contrast meets WCAG 2.1 AA standards, and images include descriptive alt text. The sitemap uses SVG with proper roles and labels for screen readers.

Challenges and Lessons Learned

Coordinating the sitemap SVG layout to reflect all connections clearly was challenging but improved my understanding of scalable vector graphics. Maintaining consistent navigation and footer links across pages enhanced usability and user experience.
